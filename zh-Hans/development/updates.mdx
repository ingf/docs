<Update label="2025-02-14" description="Java SDK 发布">
  - 我们很高兴地宣布,由 Spring AI at VMware Tanzu 开发的 Java SDK 现在成为了 MCP 的官方 [Java SDK](https://github.com/modelcontextprotocol/java-sdk)。
    这将与我们现有的 Kotlin SDK 一起加入到我们不断增长的支持语言列表中。
    Spring AI 团队将作为 Model Context Protocol 组织的重要组成部分维护该 SDK。
    我们很高兴欢迎他们加入 MCP 社区！
</Update>

<Update label="2025-01-27" description="Python SDK 1.2.1">
  - MCP Python SDK 的 [1.2.1](https://github.com/modelcontextprotocol/python-sdk/releases/tag/v1.2.1) 版本已发布，
    带来了重要的稳定性改进和错误修复。
</Update>

<Update label="2025-01-18" description="SDK 和服务器改进">
  - [TypeScript SDK](https://github.com/modelcontextprotocol/typescript-sdk) 中新增简化的 express 风格 API
  - 在[客户端页面](https://modelcontextprotocol.io/clients)新增 8 个新客户端
</Update>

<Update label="2025-01-03" description="SDK 和服务器改进">
  - [Python SDK](https://github.com/modelcontextprotocol/python-sdk) 中新增 FastMCP API
  - [servers 仓库](https://github.com/modelcontextprotocol/servers)中新增 Docker 化的 MCP 服务器
</Update>

<Update label="2024-12-21" description="Kotlin SDK 发布">
  - Jetbrains 发布了 MCP 的 Kotlin SDK！
  - 查看[此仓库](https://github.com/modelcontextprotocol/kotlin-sdk/tree/main/samples/kotlin-mcp-server)获取 MCP Kotlin 服务器示例
</Update>
