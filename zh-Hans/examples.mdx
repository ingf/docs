本页面展示了官方维护的 MCP 服务器实现。这些服务器可以直接使用，也可以作为构建自己服务器的参考。

## 文件系统服务器

提供对本地文件系统的访问。

```bash
npm install -g @modelcontextprotocol/server-filesystem
npx @modelcontextprotocol/server-filesystem ~/Documents
```

### 功能

- 将目录暴露为资源
- 支持文件读取和目录列表
- 文件变更通知
- 文件类型检测
- 路径验证和安全检查

### 资源

- `file://{path}` - 访问文件内容
- `dir://{path}` - 列出目录内容

### 工具

- `listDirectory` - 列出目录内容
- `searchFiles` - 搜索文件
- `getFileInfo` - 获取文件元数据

## Web 浏览器服务器

提供对 Web 浏览器功能的访问。

```bash
npm install -g @modelcontextprotocol/server-browser
npx @modelcontextprotocol/server-browser
```

### 功能

- 网页截图
- DOM 访问
- 网页内容提取
- Cookie 和会话管理
- JavaScript 执行

### 资源

- `page://{url}` - 访问网页内容
- `screenshot://{url}` - 获取网页截图
- `dom://{url}` - 访问 DOM 结构

### 工具

- `screenshot` - 捕获网页截图
- `extract` - 提取网页内容
- `evaluate` - 执行 JavaScript

## 全功能服务器

组合了所有官方服务器功能的完整实现。

```bash
npm install -g @modelcontextprotocol/server-everything
npx @modelcontextprotocol/server-everything
```

### 功能

包含以下所有功能：
- 文件系统访问
- Web 浏览器功能
- 数据库连接
- API 集成
- 工具执行
- 提示处理

### 资源

- 所有文件系统资源
- 所有浏览器资源
- `db://{connection}` - 数据库访问
- `api://{endpoint}` - API 端点

### 工具

- 所有文件系统工具
- 所有浏览器工具
- 数据库查询工具
- API 请求工具
