以下是支持 Model Context Protocol 的客户端列表。

## 官方客户端

### Claude Desktop

[Claude Desktop](https://claude.ai/desktop) 是一个原生的桌面应用程序,用于与 Claude 交互。它支持:

- 从本地文件和应用程序加载上下文
- 通过 MCP 服务器扩展功能
- 跨平台支持 (macOS, Windows, Linux)

### MCP Inspector

[MCP Inspector](https://github.com/modelcontextprotocol/inspector) 是一个用于测试和调试 MCP 服务器的开发者工具。它提供:

- 交互式命令行界面
- 服务器功能探索
- 工具和资源测试
- 提示模板执行

## 社区客户端

### VSCode MCP

[VSCode MCP](https://marketplace.visualstudio.com/items?itemName=mcp.vscode-mcp) 是一个 Visual Studio Code 扩展,它:

- 将 Claude 集成到您的 IDE 中
- 支持代码分析和生成
- 提供上下文感知的建议
- 启用工作区范围的上下文

### IntelliJ MCP

[IntelliJ MCP](https://plugins.jetbrains.com/plugin/12345-mcp) 为 JetBrains IDE 提供了类似的功能:

- 智能代码补全
- 代码审查和建议
- 文档生成
- 测试用例生成

### Emacs MCP

[Emacs MCP](https://github.com/username/emacs-mcp) 为 Emacs 编辑器添加了 MCP 支持:

- 与 Claude 的直接集成
- 自定义命令和工作流
- Org-mode 集成
- 可扩展的插件系统

## 添加您的客户端

如果您开发了支持 MCP 的客户端,请[提交 PR](https://github.com/modelcontextprotocol/website) 将其添加到此列表中。

<Note>
  所有列出的客户端都必须遵守 [MCP 规范](https://modelcontextprotocol.ai/specification)。
</Note>
