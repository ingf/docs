<Update label="2025-02-14" description="Java SDKリリース">
  - Spring AIがVMware Tanzuで開発したJava SDKが、MCPの公式[Java SDK](https://github.com/modelcontextprotocol/java-sdk)としてリリースされました。
    これは、既存のKotlin SDKに加わり、サポートされている言語のリストを拡大しています。
    Spring AIチームは、Model Context Protocol組織の一部としてSDKを維持します。MCPコミュニティに彼らを迎えることを嬉しく思います！
</Update>

<Update label="2025-01-27" description="Python SDK 1.2.1">
  - MCP Python SDKのバージョン[1.2.1](https://github.com/modelcontextprotocol/python-sdk/releases/tag/v1.2.1)がリリースされ、重要な安定性の向上とバグ修正が行われました。
</Update>
<Update label="2025-01-18" description="SDKとサーバーの改善">
  - [TypeScript SDK](https://github.com/modelcontextprotocol/typescript-sdk)における簡潔で表現力豊かなAPI
  - [クライアントページ](https://modelcontextprotocol.io/clients)に8つの新しいクライアントを追加
</Update>

<Update label="2025-01-03" description="SDKとサーバーの改善">
  - [Python SDK](https://github.com/modelcontextprotocol/python-sdk)におけるFastMCP API
  - [サーバーレポ](https://github.com/modelcontextprotocol/servers)におけるDocker化されたMCPサーバー
</Update>

<Update label="2024-12-21" description="Kotlin SDKリリース">

  - MCP Kotlinサーバーのサンプルについては、[このリポジトリ](https://github.com/modelcontextprotocol/kotlin-sdk/tree/main/samples/kotlin-mcp-server)をチェックしてください
</Update>
