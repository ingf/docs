このページでは、Model Context Protocol (MCP) の能力と多様性を示すさまざまなサーバーを紹介します。これらのサーバーは、LLMがツールやデータソースに安全にアクセスすることを可能にします。

## リファレンス実装

これらの公式リファレンスサーバーは、MCPのコア機能とSDKの使用法を示しています：

### データとファイルシステム
- **[Filesystem](https://github.com/modelcontextprotocol/servers/tree/main/src/filesystem)** - 設定可能なアクセス制御を備えた安全なファイル操作
- **[PostgreSQL](https://github.com/modelcontextprotocol/servers/tree/main/src/postgres)** - スキーマ検査機能を備えた読み取り専用のデータベースアクセス
- **[SQLite](https://github.com/modelcontextprotocol/servers/tree/main/src/sqlite)** - データベースの操作とビジネスインテリジェンス機能
- **[Google Drive](https://github.com/modelcontextprotocol/servers/tree/main/src/gdrive)** - Google Driveのファイルアクセスと検索機能

### 開発ツール
- **[Git](https://github.com/modelcontextprotocol/servers/tree/main/src/git)** - Gitリポジトリの読み取り、検索、操作のためのツール
- **[GitHub](https://github.com/modelcontextprotocol/servers/tree/main/src/github)** - リポジトリ管理、ファイル操作、GitHub API統合
- **[GitLab](https://github.com/modelcontextprotocol/servers/tree/main/src/gitlab)** - プロジェクト管理を可能にするGitLab API統合
- **[Sentry](https://github.com/modelcontextprotocol/servers/tree/main/src/sentry)** - Sentry.ioからの問題の取得と分析

### Webとブラウザの自動化
- **[Brave Search](https://github.com/modelcontextprotocol/servers/tree/main/src/brave-search)** - BraveのSearch APIを使用したWebおよびローカル検索
- **[Fetch](https://github.com/modelcontextprotocol/servers/tree/main/src/fetch)** - LLM使用に最適化されたWebコンテンツの取得と変換
- **[Puppeteer](https://github.com/modelcontextprotocol/servers/tree/main/src/puppeteer)** - ブラウザの自動化とWebスクレイピング機能

### 生産性とコミュニケーション
- **[Slack](https://github.com/modelcontextprotocol/servers/tree/main/src/slack)** - チャンネル管理とメッセージング機能
- **[Google Maps](https://github.com/modelcontextprotocol/servers/tree/main/src/google-maps)** - 位置情報サービス、方向、場所の詳細
- **[Memory](https://github.com/modelcontextprotocol/servers/tree/main/src/memory)** - ナレッジグラフベースの永続的なメモリシステム

### AIと専門ツール
- **[EverArt](https://github.com/modelcontextprotocol/servers/tree/main/src/everart)** - さまざまなモデルを使用したAI画像生成
- **[Sequential Thinking](https://github.com/modelcontextprotocol/servers/tree/main/src/sequentialthinking)** - 思考シーケンスによる動的な問題解決
- **[AWS KB Retrieval](https://github.com/modelcontextprotocol/servers/tree/main/src/aws-kb-retrieval-server)** - Bedrock Agent Runtimeを使用したAWS Knowledge Baseからの取得

## 始めましょう

### リファレンスサーバーの使用

TypeScriptベースのサーバーは`npx`で直接使用できます：

```bash
npx -y @modelcontextprotocol/server-memory
```

Pythonベースのサーバーは`uvx`（推奨）または`pip`で使用できます：

```bash
# uvxを使用
uvx mcp-server-git

# pipを使用
pip install mcp-server-git
python -m mcp_server_git
```

### Claudeでの設定

MCPサーバーをClaudeで使用するには、設定に追加します：

```json
{
  "mcpServers": {
    "memory": {
      "command": "npx",
      "args": ["-y", "@modelcontextprotocol/server-memory"]
    },
    "filesystem": {
      "command": "npx",
      "args": ["-y", "@modelcontextprotocol/server-filesystem", "/path/to/allowed/files"]
    },
    "github": {
      "command": "npx",
      "args": ["-y", "@modelcontextprotocol/server-github"],
      "env": {
        "GITHUB_PERSONAL_ACCESS_TOKEN": "<YOUR_TOKEN>"
      }
    }
  }
}
```
