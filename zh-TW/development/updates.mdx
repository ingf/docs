<Update label="2025-02-14" description="Java SDK 發布">
  - 我們很高興宣布由 Spring AI at VMware Tanzu 開發的 Java SDK 現在是 MCP 的官方 [Java SDK](https://github.com/modelcontextprotocol/java-sdk)。
    這與我們現有的 Kotlin SDK 一起加入了我們不斷增長的支持語言列表。
    Spring AI 團隊將維護該 SDK，使其成為 Model Context Protocol 組織不可或缺的一部分。
    我們非常高興歡迎他們加入 MCP 社群！
</Update>

<Update label="2025-01-27" description="Python SDK 1.2.1">
  - MCP Python SDK 的 [1.2.1](https://github.com/modelcontextprotocol/python-sdk/releases/tag/v1.2.1) 版本已發布，
    帶來了重要的穩定性改進和錯誤修復。
</Update>
<Update label="2025-01-18" description="SDK 和伺服器改進">
  - [TypeScript SDK](https://github.com/modelcontextprotocol/typescript-sdk) 中簡化的、類似 express 的 API
  - 在[客戶端頁面](https://modelcontextprotocol.io/clients) 中新增了 8 個客戶端
</Update>

<Update label="2025-01-03" description="SDK 和伺服器改進">
  - [Python SDK](https://github.com/modelcontextprotocol/python-sdk) 中的 FastMCP API
  - [伺服器儲存庫](https://github.com/modelcontextprotocol/servers) 中 Docker 化的 MCP 伺服器
</Update>
