此頁面展示了各種 Model Context Protocol (MCP) 伺服器，這些伺服器示範了該協定的功能和多樣性。這些伺服器使大型語言模型 (LLMs) 能夠安全地存取工具和資料來源。

## Reference implementations

這些官方參考伺服器示範了核心 MCP 功能和 SDK 用法：

### Data and file systems
- **[Filesystem](https://github.com/modelcontextprotocol/servers/tree/main/src/filesystem)** - 具有可配置存取控制的安全檔案操作
- **[PostgreSQL](https://github.com/modelcontextprotocol/servers/tree/main/src/postgres)** - 具有結構描述檢查功能的唯讀資料庫存取
- **[SQLite](https://github.com/modelcontextprotocol/servers/tree/main/src/sqlite)** - 資料庫互動和商業智慧功能
- **[Google Drive](https://github.com/modelcontextprotocol/servers/tree/main/src/gdrive)** - Google Drive 的檔案存取和搜尋功能

### Development tools
- **[Git](https://github.com/modelcontextprotocol/servers/tree/main/src/git)** - 讀取、搜尋和操作 Git 儲存庫的工具
- **[GitHub](https://github.com/modelcontextprotocol/servers/tree/main/src/github)** - 儲存庫管理、檔案操作和 GitHub API 整合
- **[GitLab](https://github.com/modelcontextprotocol/servers/tree/main/src/gitlab)** - GitLab API 整合，實現專案管理
- **[Sentry](https://github.com/modelcontextprotocol/servers/tree/main/src/sentry)** - 從 Sentry.io 檢索和分析問題

### Web and browser automation
- **[Brave Search](https://github.com/modelcontextprotocol/servers/tree/main/src/brave-search)** - 使用 Brave Search API 進行網路和本地搜尋
- **[Fetch](https://github.com/modelcontextprotocol/servers/tree/main/src/fetch)** - 針對 LLM 使用進行優化的網路內容獲取和轉換
- **[Puppeteer](https://github.com/modelcontextprotocol/servers/tree/main/src/puppeteer)** - 瀏覽器自動化和網頁抓取功能

### Productivity and communication
- **[Slack](https://github.com/modelcontextprotocol/servers/tree/main/src/slack)** - 頻道管理和訊息傳遞功能
- **[Google Maps](https://github.com/modelcontextprotocol/servers/tree/main/src/google-maps)** - 位置服務、路線和地點詳細資訊
- **[Memory](https://github.com/modelcontextprotocol/servers/tree/main/src/memory)** - 基於知識圖的持久記憶體系統

### AI and specialized tools
- **[EverArt](https://github.com/modelcontextprotocol/servers/tree/main/src/everart)** - 使用各種模型進行 AI 圖像生成
- **[Sequential Thinking](https://github.com/modelcontextprotocol/servers/tree/main/src/sequentialthinking)** - 通過思維序列進行動態問題解決
- **[AWS KB Retrieval](https://github.com/modelcontextprotocol/servers/tree/main/src/aws-kb-retrieval-server)** - 使用 Bedrock Agent Runtime 從 AWS Knowledge Base 檢索

## Getting started

### Using reference servers

基於 TypeScript 的伺服器可以直接與 `npx` 一起使用：

```bash
npx -y @modelcontextprotocol/server-memory
```

基於 Python 的伺服器可以與 `uvx`（推薦）或 `pip` 一起使用：

```bash
# Using uvx
uvx mcp-server-git

# Using pip
pip install mcp-server-git
python -m mcp_server_git
```

### Configuring with Claude

若要將 MCP 伺服器與 Claude 一起使用，請將其新增至您的配置：

```json
{
  "mcpServers": {
    "memory": {
      "command": "npx",
      "args": ["-y", "@modelcontextprotocol/server-memory"]
    },
    "filesystem": {
      "command": "npx",
      "args": ["-y", "@modelcontextprotocol/server-filesystem", "/path/to/allowed/files"]
    },
    "github": {
      "command": "npx",
      "args": ["-y", "@modelcontextprotocol/server-github"],
      "env": {
        "GITHUB_PERSONAL_ACCESS_TOKEN": "<YOUR_TOKEN>"
      }
    }
  }
}
```
